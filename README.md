# Black Magic Probe for RP2040
# WORK IN PROGRESS

Attempt to port firmware for the Black Magic probe to RP2040.

# CURRENTLY WORKING
- usb serial communication
- working gdb server

# TO-DO

# How to compile

2. run ```make PROBE_HOST=rp2040```
